Genshin Speedrun Contest: Dijkstra's Algorithm Solver

CS 5800

Usage

  • While Draw point is selected, click anywhere in the map to create a point.
  • To draw an edge between two points, select the Draw edge radio button, then click on the points you want to connect.
  • To change the cost or point label, click on the cost or the label while Set cost or label radio button is selected.
  • Finally, to run the algorithm, select Set Start then click on the starting point.
